Generative Geometry

Space Station Fractal

Start with one horizontal segment of length L. Bisect it with a perpendicular segment of length F L, then recurse on the four resulting half-segments. The canvas below redraws that rule directly, with depth and scale controls so you can explore how the structure grows.

Rule Each segment spawns a centered perpendicular cross and four smaller child segments.
Default Ratio The perpendicular branch uses F = 0.5, so a segment of length L produces vertical children of length F L / 2.
Rendering The drawing auto-fits the viewport and rescales when you change parameters or resize the page.
Segments drawn: 0 Cycle 1 of 6 Starting rule: horizontal(L) -> perpendicular(F * L) -> recurse on 4 half-segments
